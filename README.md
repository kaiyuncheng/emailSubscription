# Email Subscription Demo 
# Email 訂閱系統

https://desolate-stream-72902.herokuapp.com/

+ 使用Node.js, Express, EJS, bodyParser
+ Server 架設於 Heroku 上
+ 用[Mailchimp](https://mailchimp.com/) API 接收收集回傳個人資料與信箱，作為未來發送電子信使用 
+ 表單驗證 - Email格式和所有欄位不能留空
